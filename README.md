# 🏃 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
```
git clone https://github.com/front-end-by-rimantas/41-grupe-ufo.git
```
2. Install NPM packages
```
npm i
```
or
```
npm install
```
3. Run the server
```
npm run dev
```
